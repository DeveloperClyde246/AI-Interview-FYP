# AI-Interview-FYP-Fullstack

## to setup:
'''
git clone https://github.com/DeveloperClyde246/AI-Interview-FYP-Backend.git

cd AI-Interview-FYP-Backend
'''
### Install the packages
'''
npm install
'''

### Start Backend
Open new terminal
''' 
npm run devStart 
'''

### Start Frontend
Open another terminal

'''
cd client
npm run dev
'''
